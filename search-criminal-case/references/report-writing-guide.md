# 报告写作指南

> **加载时机**：Phase 5（并行写作）执行时读取
> **用途**：提供报告各章节的详细写作指南、典型类案评析模板和量刑差异归因分析方法

---

## 一、各章节写作指南

### 1.1 一、案件概述

**1.1 基本案情**
- 客观描述用户提交的案件事实，不加主观评价
- 篇幅控制在 300-500 字
- 需包含：时间、地点、行为人、行为方式、侵害对象、危害后果
- 涉及技术细节的，用通俗语言解释

**1.2 案情要素提取**
- 以表格形式呈现 Phase 1 提取的结构化要素
- 缺失字段标注"未提及"
- 表格后附"要素缺失提示"段落

### 1.2 二、罪名分析

**2.1 候选罪名及匹配**
- 列出所有候选罪名，标注匹配度（高/中/低）
- 每个罪名附匹配理由（要素与要件逐条对照）
- 如有竞合关系，在此节点点明

**2.2 要件对比分析**
- 以表格形式呈现：要件 × 本案要素 × 罪名1要件 × 是否符合 × 罪名2要件 × 是否符合
- 不符合的要件需说明原因和潜在争议

**2.3 竞合关系判断**
- 如存在多个罪名，分析竞合关系
- 给出建议适用罪名及理由
- 引用竞合关系判断规则（见 references/crime-elements.md）

### 1.3 三、类案检索说明

**3.1 检索策略**
- 说明使用的检索通道（语义/关键词/本地）
- 列出检索关键词表达式
- 说明筛选条件（时间、审级、法院级别等）

**3.2 检索结果概览**
- 检索案件总数
- 去重后案件数
- 质量筛选后有效案件数
- 指导性/典型案例数

### 1.4 四、类案统计分析

**4.1 案件总览**
- 有效案件总数
- 时间分布（年份）
- 地域分布（省份/法院）
- 审级分布

**4.2 量刑分布**
- 按量刑档次分组的统计表
- 含案件数、占比、典型案例案号
- 缓刑率单独统计

**4.3 要素交叉分析**
- 按核心要素（数据规模/经济损失/行为类型）分组的量刑均值/中位数/区间
- 以表格呈现，突出与本案要素最接近的分组

**4.4 同案不同判现象识别**
- 找出要素相似但量刑差异显著的案件对
- 列出差异最大的案件对比
- 初步标注可能的归因方向

### 1.5 五、典型类案评析

每个案件按"典型类案评析写作模板"（见下文）撰写。

### 1.6 六、量刑差异归因

**6.1 法定量刑情节差异**
- 比较各案件中法定从轻/从重情节的认定和适用
- 自首、坦白、立功、累犯等情节对量刑的影响

**6.2 酌定量刑情节差异**
- 退赔退缴、认罪认罚、初犯等情节的适用差异
- 被告人个人情况（年龄、职业、前科）对量刑的影响

**6.3 地域/审级差异**
- 不同地区法院的量刑倾向差异
- 一审与二审的量刑调整方向

**6.4 裁判逻辑差异**
- 法院对同一行为定性的差异
- 对"情节严重"/"后果严重"认定标准的差异
- 对技术手段危害性评价的差异

### 1.7 七、对本案的启示

**7.1 辩护策略建议**
- 基于类案统计的量刑预期区间
- 可主张的从轻情节
- 定性争议点的辩护方向

**7.2 类案参考要点**
- 最具参考价值的 2-3 个类案
- 参考价值的具体体现

**7.3 风险提示**
- 量刑上限风险
- 定性争议风险
- 证据要求风险

- 裁判时间：[年月]

- 主刑：[有期徒刑X年/拘役X月]

|------|------|------|----------|
| 行为类型 | ... | ... | 相同/不同 |
| 数据规模 | ... | ... | ... |
| 后果程度 | ... | ... | ... |
| 从轻情节 | ... | ... | ... |

## 三、量刑差异归因分析方法

### 3.1 量化对比法

1. 选取要素相似（行为类型、数据规模、后果程度三项一致）的案件组
2. 计算组内量刑的均值、中位数、标准差
3. 偏离均值±1个标准差以上的案件标记为"异常量刑"
4. 分析异常量刑案件的特殊情节

### 3.2 要素控制法

1. 固定其他要素不变，改变单一要素
2. 观察该要素对量刑的影响方向和幅度
3. 常见控制变量：数据规模、经济损失、自首、退赔退缴、认罪认罚

### 3.3 裁判逻辑分析法

1. 提取裁判文书中的说理段落
2. 对比不同法院对同一要件的认定逻辑
3. 识别"同案不同判"的裁判逻辑差异点
4. 常见差异点：行为危害性评价、情节严重性认定、缓刑适用条件

## 四、写作注意事项

1. **客观中立**：报告正文保持客观描述，评析部分基于数据支撑，不主观臆断
2. **数据准确**：所有统计数字必须来自实际检索结果，百分比合计为100%
3. **法条引用**：引用法条时使用准确条款项序号（如"第285条第2款"而非"第285条"）
4. **案号格式**：案号使用完整格式（如"(2023)京01刑终123号"）
5. **避免绝对化**：使用"倾向于""可能""通常"等限定词，不使用"必然""一定"等绝对化表述
6. **篇幅控制**：总篇幅30-50页，典型类案评析每件1-2页，统计分析部分5-8页
