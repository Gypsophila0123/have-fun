# 格式辅助规范


> 本文件定义计算机犯罪类案检索报告的排版与格式输出标准。


---

| 项目 | 规格 |

| 下边距 | 2.54 cm |

| 右边距 | 3.07 cm |

## 二、标题字体矩阵


| 级别 | Markdown | 字体 | 字号 | 加粗 | 行距 | 说明 |
|------|----------|------|------|------|------|------|
| 报告标题 | `# 标题` | 黑体 | 小三(15pt) | 是 | 1.25 | 全文仅一个 |
| 一级章节 | `## 一、XXX` | 黑体 | 小三(15pt) | 是 | 1.25 | 如：案件概述、罪名分析 |
| 二级子节 | `### 1.1 XXX` | 黑体 | 四号(14pt) | 是 | 1.25 | 如：基本案情、法条依据 |
| 三级条目 | `#### （一）XXX` | 黑体 | 小四(12pt) | 是 | 1.25 | 如：构成要件符合性 |
| 四级细目 | `##### 1. XXX` | 宋体 | 小四(12pt) | 加粗 | 1.25 | 如：行为主体、行为对象 |
| 五级说明 | `###### (1) XXX` | 宋体 | 小四(12pt) | 否 | 1.25 | 单要素说明 |
| 列表项 | `- XXX` | 宋体 | 小四(12pt) | 否 | 1.25 | 普通列表 |


### 使用说明


1. **标题深度原则**：常规报告使用至第四级即可；复杂分析允许展开至第五级。
2. **章节编号**：一级使用中文数字（一、二、三），二级使用阿拉伯数字（1.1、1.2），三级使用带括号中文数字（（一）、（二））。
3. **结论标记**：每一分析步骤结尾使用 ✓（肯定）或 ✗（否定）标记结论。

## 三、引用块样式


- **字体**：楷体
- **字号**：小四 (12pt)
- **样式**：左侧灰色竖线（Markdown blockquote `>`）
- **内容结构**：法条全文 + 校验状态标签


### 校验状态标签


```
> 《中华人民共和国刑法》第二百八十五条　违反国家规定，侵入国家事务、国防建设、尖端科学技术领域的计算机信息系统的，处三年以下有期徒刑或者拘役……
> [校验2026-07-15, 现行有效]
```


三种状态：
- `[校验YYYY-MM-DD, 现行有效]` — 校验通过
- `[⚠ 校验失败]` — MCP未能检索到匹配法条
- `[⚠ 语义差异]` — 检索到法条但文本不完全一致

## 四、表格样式规范


### 基础样式


| 项目 | 规格 |
|------|------|
| 字体 | 宋体 |
| 字号 | 小四(12pt) |
| 边框 | 全边框（1pt黑色实线） |
| 对齐 | 表头居中，内容左对齐（数字类右对齐） |
| 行距 | 1.25 倍 |
| 单元格边距 | 左右0.5cm，上下0.2cm |


### 特殊表格类型


#### 法条对照表
用于展示罪名与法条的对应关系：
- 表头背景色：浅灰色（RGB: 240, 240, 240）
- 表格宽度：100%


#### 相似度比对表
用于展示案例多维相似度对比：
- 表头背景色：浅蓝色（RGB: 230, 240, 255）
- 表格宽度：100%
- 权重列使用加粗字体


#### 统计分析表
用于展示类案统计数据：
- 表头背景色：浅绿色（RGB: 230, 255, 230）
- 表格宽度：100%
- 数值列右对齐

## 五、段首缩进


- 正文段落首行缩进：使用 U+3000（全角空格）× 2
- Markdown源文件中直接输入两个全角空格
- 引用块内部同样适用

## 六、来源标识符


仅允许以下四种脚注来源标识符：


| 标识符 | 含义 |
|--------|------|
| `[评注]` | 法律评注/教科书引用 |
| `[案例]` | 司法案例引用 |
| `[联网]` | 网络检索来源 |
| `[KB:]` | 知识库检索来源（冒号后接KB名称） |


### 脚注格式


```
正文内容。^[来源标识符 作者/案号, 具体出处, 页码/段落]
```

## 七、文件命名规则


```
检索报告-[YYYY-MM-DD].docx
检索报告-[YYYY-MM-DD]-2.docx
```


- 同一天多次检索自动递增序号
- 保留所有历史报告


---


## 八、风险提醒样式


```
> ⚠️ **风险提醒**：本报告由系统自动生成，基于本地案例库检索结果，仅供参考，不构成任何法律意见或建议。具体案件请咨询专业律师。
```


- **字体**：楷体
- **字号**：小四(12pt)
- **位置**：报告开头（标题下方）

```

```


### 转换检查清单


- [ ] 标题层级正确映射为Word标题样式
- [ ] 引用块转换为楷体格式
- [ ] 页边距/行距与本规范一致
- [ ] 页码居中，首页不显示
- [ ] 段首缩进保留
- [ ] 表格样式符合规范（边框、背景色）

## 十、其他排版规则


1. **中西文间距**：中文与英文/数字之间自动添加半角空格
2. **法条编号**：使用中文数字（第二百八十五条），不使用阿拉伯数字
3. **案例编号**：保留原始案号格式
4. **结论标记**：使用 ✓（肯定）或 ✗（否定）标记分析结论
